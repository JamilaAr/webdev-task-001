  • HTML: The standard markup language used to create web pages.
  • CSS: Stylesheet language used to describe the presentation of a document written in HTML.
  • JavaScript: Programming language that allows you to create dynamic content on web pages.
  • DOM: Document Object Model, a programming interface for HTML and XML documents.
  • API: Application Programming Interface, a set of functions for building software applications.
